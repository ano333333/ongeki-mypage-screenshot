Ongeki MyPage Screenshot Bookmarklet

使い方

  1. 下のボタンをクリックして、ブックマークレットのコードをコピーしてください
  2. ブラウザでブックマークを新規作成し、URLにコピーしたコードを貼り付けてください
  3. オンゲキのマイページで、作成したブックマークレットをクリックしてください
  4. スクリーンショットが自動的に撮影されます

以下はPC版Chromeの場合の画像です。

説明

このブックマークレットは、オンゲキのマイページでスクリーンショットをイイ感じに撮影するためのツールです。

対応ページ

  • プレイ履歴(https://ongeki-net.com/ongeki-mobile/record/playlog/)
  • 楽曲別ランキング(https://ongeki-net.com/ongeki-mobile/ranking/musicRankingDetail)

使い方

プレイ履歴

ブックマークをクリックしてください。「キャプチャする」のボタンが左上に出たら、キャプチャに含めたいリザルトをクリック・タップで選択してください。「キャプチャする」ボタンを押すと、選択したリザルトをまとめた画像をダウンロードします。

曲別ランキング

ブックマークをクリックしてください。「キャプチャする」のボタンが左上に出たら、ランキングのキャプチャに含めたい行をクリック・タップで選択してください。「キャプチャする」ボタンを押すと、選択した行をまとめた画像をダウンロードします。

iOS端末での注意

作成した画像を新規ウィンドウで開く際、それがブラウザによりブロックされることがあります。「常に表示」をクリックし再度お試しください。
